# Mysbah Start Here

This website updates automatically.

You only need GitHub.

Repository:

```text
https://github.com/Assembler-Fourier/ugcbymysbah
```

## Change The Main Picture

1. Open:

```text
public/images
```

2. Replace this file:

```text
00-main-profile-photo.jpeg
```

3. Commit to `main`.
4. Wait for Vercel to finish.

## Change The Travel Picture

Replace:

```text
public/images/00-travel-profile-photo.jpeg
```

## Add Portfolio Videos

Open:

```text
public/videos
```

Upload MP4 files using these exact names:

```text
01-lifestyle-creator-intro.mp4
02-beauty-demo-routine.mp4
03-travel-nomad-story.mp4
04-wellness-testimonial-hook.mp4
05-skincare-texture-shot.mp4
06-unboxing-friend-rec-energy.mp4
```

Each name goes to its matching card.

## Important

- Keep the file names exactly the same.
- Use JPEG for photos.
- Use MP4 for videos.
- Keep videos compressed.
- Commit to `main`.
- Vercel publishes the website automatically.
