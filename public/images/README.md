# Image Slots

Replace these exact files to change the website photos.

| File name | Where it appears |
| --- | --- |
| `00-main-profile-photo.jpeg` | Top hero photo, social preview image, Lifestyle portfolio card |
| `00-travel-profile-photo.jpeg` | Travel portfolio card |

How to replace a photo:

1. Prepare a JPEG image.
2. Rename it to the exact file name above.
3. Upload it into this `public/images` folder on GitHub.
4. Commit to `main`.
5. Vercel auto-deploys the website.

If GitHub says the file already exists, delete the old file first, then upload the new file with the exact same name.
